# 4강 1조 프로젝트 관리 툴

## 📋 프로젝트 개요

- 조직 맞춤형 프로젝트/사용자/그룹/권한/일감 설정을 통합 관리할 수 있는

- 독립형 프로젝트 관리 툴을 직접 구축하고자 하였습니다.

- 이를 통해 조직 맞춤형 업무 프로세스의 정립과, 관리자의 운영 효율이 향상되는 기대효과가 예상됩니다.
